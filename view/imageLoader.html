About this Project

The reason why I made this Image Loader is to gain a better understanding of how image files work, specifically regarding the decoding of PNG, JPEG, and BMP formats. To achieve this, I am building this project from scratch without relying on any external dependencies. For instance, I had to implement the zlib decoding for PNG's data, on my own.

By including the header in C++, this image loader can load PNG, JPEG, and BMP images by inputting the file path.

To demonstrate, I've created a demo program. After execution, open any image by clicking file->open.
